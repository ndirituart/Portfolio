Data Visualization and Dashboard Creation

Difficulty: Relatively Easy

Approach

  • Cleaning and sorting data.
  • Creating visualizations using Excel, Power BI and Tableau.
  • Testing to ensure visualizations are clear and actionable.

// Key Lesson Learned

Understanding business needs so that visualizations are meaningful and actionable.

Challenge: Learning new Tools and Techniques

It was my first time using Tableau that was definitely interesting to learn. I knew how to import data, create visualizations and build dashboards from PowerBi and used those principles. However, I have used Power BI before but a while back and found the multiple interfaces and dashboards confusing to remember and follow through.

Problem Statement

The target application allowed session fixation via a URL parameter, leading to account takeover.

Approach & Tools

Intercepted traffic with Burp Suite to observe session token generation. Used FFUF for directory discovery.

Outcome

Successfully fixed the vulnerability by implementing secure session management and invalidating tokens after use.

Key Lessons

  • Always rotate session IDs upon login.
  • URL-based tokens are highly insecure.
